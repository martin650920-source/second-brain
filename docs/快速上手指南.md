## 🖥️ 周一安裝（Windows）

**開 PowerShell（系統管理員）**
```powershell
# 進到 second-brain 資料夾
cd "D:\GOOGLE_DRIVE_SYNC\second-brain"

# 執行安裝腳本
pwsh -ExecutionPolicy Bypass -File setup\setup-windows.ps1
```

**安裝過程會問你：**
- 偵測到舊版 symlink？→ 輸入 `Y` 更新
- 啟用 Secure Mode？→ 輸入 `Y`（建議）
- 其他詢問 → 看提示選擇

**確認安裝成功**
```
看到以下代表成功：
  C:\Users\你\\.second-brain -> D:\GOOGLE_DRIVE_SYNC\second-brain  ✓
  C:\Users\你\\.claude\CLAUDE.md -> ...adapters\claude\CLAUDE.md   ✓
  C:\Users\你\\.gemini\GEMINI.md -> ...adapters\gemini\GEMINI.md   ✓
  C:\Users\你\\.codex\AGENTS.md  -> ...adapters\codex\AGENTS.md    ✓
```

## ✍️ 首次使用要填的資料

安裝完後，這幾個檔案是空的，按需填入：

**1. 工作背景**
```
檔案：~/.second-brain/contexts/work/context.md
填什麼：公司背景、常用系統（GitLab URL 等）
```

**2. 生活背景**
```
檔案：~/.second-brain/contexts/life/context.md
填什麼：個人習慣、常問 AI 的生活問題類型
```

**3. MCP 工具（有需要再填）**
```
檔案：~/.second-brain/contexts/work/mcp.json
參考：contexts/work/mcp.template.json
```

## ➕ 新增工作專案

```bash
# 1. 複製範本
cp ~/.second-brain/projects/_template.md \
   ~/.second-brain/projects/新專案名.md

# 2. 填入內容（架構、指令、術語）
# 用編輯器打開填寫

# 3. 在 .gitignore 加入（避免公司 IP 外流）
echo "/projects/新專案名.md" >> ~/.second-brain/.gitignore

# 4. 在 context-loader 加入偵測規則
# 編輯 ~/.second-brain/skills/context-loader/SKILL.md
# 在 Step 5 的表格加一行 Marker Files
```
